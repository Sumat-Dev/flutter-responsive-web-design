## 🛠️ Getting Started

### Installation
1. Clone the repository
```bash
git clone https://github.com/Sumat-Dev/flutter-responsive-web-design.git
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the application
```bash
flutter run
```
